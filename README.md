# UrbanEye-API-Smart-Traffic-Management-using-Computer-Vision

UrbanEye is an AI-powered traffic monitoring system built using **FastAPI** and **YOLO**.
It provides real-time detection of traffic violations such as helmet absence, multiple riders,
and license plate presence through REST APIs.

## 🚀 Features
- YOLO-based object detection
- Image & video analysis
- REST API with Swagger UI
- JSON-based structured output
- Scalable API-first architecture

## 🛠 Tech Stack
- Python
- FastAPI
- YOLO (Ultralytics)
- OpenCV
- Torch
